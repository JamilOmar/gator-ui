Help

To have Dev Star function properly, administration of the organization has to do the followings:
  • Install webhooks
  • Hydrate Dev Star with historical data.
  • All users of Organization must login once in Dev Star.
  • Connect their Git, Azure Boards, Jira Id's.
  • Create the Org Structure.
  • Connect Git Orgs with Tenant. Contact the Dev Star Admin for this last step.

Webhook installation

You should install the webhook for the products which is used in your organization. This activity is done by the administration of your systems. This need to be done only once.
  • GitHub
  • Jira
  • TFS - Azure DevOps - Azure Boards

    • GitHub

    Without webhooks Dev Star will not update your data. Please ask the admin of your orgnization to install webhook.

    Follow these simple steps to Manually Install web hook?

    • Goto your organization page on the github
    • Navigate to the "Settings" tab.
    • Select the "Webhooks" option on the left menu.
    • Click "Add Webhook"
    • put 'https://gatorgithook.azurewebsites.net/api/httptrigger' in payload URL
    • Select "application/json" as the encoding type.
    • select radio button 'Let me select individual events.'
    • check 'Pull requests'
    • click on Add webhook

    Hydrate

    Once you installed the webhooks, data will start trickling in. Dev Star have an option to hydrate with some historical data right away. For that go to Admin screen, and click on "Hydrate" option. Hydrate option will work in background and pull data in. It may take couple of minutes to hours, depending upon the data size.

    With time data in Dev Star will grow and it will become more and more valuable tool for your organization.

    Connect Your Id's With Dev Star

    Log in Dev Star. Click on "Gear" Icon on the left Nav and then choose "Connect Your Id's"

    Create the Org Chart

    This step is very important. With the org chart, various security settings are implemented. Org chart is used to decide which user can see who's status reports. To create the Organization Chart. Log in Dev Star. Click on "Org" Icon on the left Nav.
